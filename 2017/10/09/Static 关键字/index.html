Static 关键字

static members belong to the class instead of a specific instance

static可用于:

  • 变量
  • 方法
  • block
  • nested class

static 变量

在一个类中没有被static声明的变量称为实例变量，实例变量伴随着类的实例而产生，同类的实例产生和结束。

但被static声明的变量（静态变量）与实例变量不同，它随着类加载时被完成初始化，并且内存中仅有一个，而且JVM也只会为它分配一次内存，同时该类的所有实例都共享这个变量（静态变量），即使实例无数个该类也就只有一个静态变量。

static 方法

在一个类中被static修饰的方法称为静态方法，静态变量不需要通过实例化的类来调用，而是可以直接通过类名来进行调用。

static block

static 修饰的block被称为静态代码块，静态代码块会随着类的加载一块执行。
